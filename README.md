# Your Energy — Health & Fitness Platform

**Your Energy** — це сучасна веб-платформа для людей, які піклуються про своє здоров'я та фізичну форму. Проєкт дозволяє переглядати вправи, фільтрувати їх за м'язовими групами та додавати улюблені тренування до списку обраного.

**Жива сторінка проєкту:** [Your Energy Demo](https://tamara-kat.github.io/goit-js-hw-your-energy-TamaraK1/)

---

## Особливості (Features)
- **Каталог вправ:** Зручний пошук та фільтрація за категоріями (Body parts, Muscles, Equipment).
- **Favorites:** Збереження обраних вправ у локальному сховищі (LocalStorage).
- **Fully Responsive:** Адаптивний дизайн для мобільних пристроїв, планшетів та десктопів (Mobile-first).
- **Quote of the Day:** Щоденна мотиваційна цитата, що завантажується через API.
- **Subscription:** Можливість підписки на оновлення через форму у футері.

---

##  Технологічний стек
- **Збірка:** [Vite](https://vitejs.dev/) — для швидкої розробки та оптимізованого білду.
- **Верстка:** HTML5 (з використанням `vite-plugin-html-inject` для модульності).
- **Стилі:** SCSS (Sass) з використанням методології **BEM (Block Element Modifier)**.
- **Мова:** JavaScript (ES6+).
- **Оптимізація:** Формат зображень WebP, адаптивні зображення (`srcset`).

## Як запустити проєкт локально

Щоб розгорнути проєкт на своєму комп'ютері, виконайте наступні кроки:

1. **Клонуйте репозиторій:**
 ```bash
   git clone [https://github.com/Tamara-Kat/goit-js-hw-your-energy-TamaraK1.git](https://github.com/Tamara-Kat/goit-js-hw-your-energy-TamaraK1.git)
```
2. **Перейдіть у папку проєкту:**
```bash
cd goit-js-hw-your-energy-TamaraK1
```
3.  **Встановіть залежності:**

```bash
npm install
```
4. **Запустіть режим розробки:**

```bash
npm run dev
```
Після цього відкрийте посилання http://localhost:5173 у вашому браузері

## Збірка проєкту (Build)
Для створення оптимізованої версії проєкту для продакшену виконайте:

```bash
npm run build
```
Готові файли будуть знаходитись у папці dist.
