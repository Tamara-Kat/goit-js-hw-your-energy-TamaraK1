# Energia ta — Platformă de sănătate și fitness

**Energia ta** este o platformă web modernă pentru persoanele cărora le pasă de sănătatea și fitness-ul lor. Proiectul vă permite să răsfoiți exerciții, să le filtrați după grupele musculare și să adăugați antrenamentele preferate în lista de favorite.


**Pagina live a proiectului:** [Demo-ul energiei tale](https://tamara-kat.github.io/goit-js-hw-your-energy-TamaraK1/)

---

## Caracteristici
- **Catalogul de exerciții:** Căutare și filtrare convenabile după categorii (Părți ale corpului, Mușchi, Echipamente).

- **Favorite:** Salvarea exercițiilor preferate în spațiul de stocare local (LocalStorage).

- **Complet responsiv:** Design adaptiv pentru dispozitive mobile, tablete și desktop-uri (Mobile-first).

- **Citatul zilei:** Citat motivațional zilnic încărcat prin API.

- **Abonament:** Posibilitatea de a vă abona la actualizări prin intermediul formularului din subsol.

---

## Stivă tehnologică
- **Constructor:** [Vite](https://vitejs.dev/) — pentru dezvoltare rapidă și compilare optimizată.
- **Layout:** HTML5 (folosind `vite-plugin-html-inject` pentru modularitate).
- **Stiluri:** SCSS (Sass) folosind metodologia **BEM (Block Element Modifier)**.
- **Limbaj:** JavaScript (ES6+).
- **Optimizare:** Format de imagine WebP, imagini responsive (`srcset`).

## Cum se rulează proiectul local

Pentru a implementa proiectul pe computer, urmați acești pași:

1. **Clonați depozitul:**
```bash
git clone [https://github.com/Tamara-Kat/goit-js-hw-your-energy-TamaraK1.git](https://github.com/Tamara-Kat/goit-js-hw-your-energy-TamaraK1.git)
```
2. **Accesați folderul proiectului:**
```bash
cd goit-js-hw-your-energy-TamaraK1
```
3. **Instalați dependențele:**

```bash
npm install
```
4. **Porniți modul de dezvoltare:**

```bash
npm run dev
```
Apoi deschideți linkul http://localhost:5173 în browser

## Construiți proiectul (Build)
Pentru a crea o versiune optimizată a proiectului pentru producție run:

```bash
npm run build
```
Fișierele finalizate vor fi în folderul dist.
