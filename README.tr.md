# Your Energy — Sağlık ve Fitness Platformu

**Your Energy**, sağlık ve fitness'ına önem veren kişiler için modern bir web platformudur. Proje, egzersizlere göz atmanıza, kas gruplarına göre filtrelemenize ve favori antrenmanlarınızı favoriler listenize eklemenize olanak tanır.

**Canlı proje sayfası:** [Your Energy Demo](https://tamara-kat.github.io/goit-js-hw-your-energy-TamaraK1/)

---

## Özellikler
- **Egzersiz Kataloğu:** Kategorilere göre (Vücut parçaları, Kaslar, Ekipman) kolay arama ve filtreleme.

- **Favoriler:** Favori egzersizlerinizi yerel depolamaya (LocalStorage) kaydetme.

- **Tamamen Duyarlı:** Mobil cihazlar, tabletler ve masaüstü bilgisayarlar için uyarlanabilir tasarım (Mobil öncelikli).

- **Günün Sözü:** API aracılığıyla yüklenen günlük motivasyonel söz.

- **Abonelik:** Altbilgideki form aracılığıyla güncellemelere abone olma özelliği.

---

## Teknoloji yığını
- **Oluşturucu:** [Vite](https://vitejs.dev/) — hızlı geliştirme ve optimize edilmiş derleme için.

- **Düzen:** HTML5 (modülerlik için `vite-plugin-html-inject` kullanılıyor).

- **Stiller:** **BEM (Block Element Modifier)** metodolojisi kullanılarak SCSS (Sass).

- **Dil:** JavaScript (ES6+).

- **Optimizasyon:** WebP resim formatı, duyarlı resimler (`srcset`).

## Projeyi Yerel Olarak Çalıştırma

Projeyi bilgisayarınıza dağıtmak için şu adımları izleyin:

1. **Depoyu klonlayın:**
```bash
git clone [https://github.com/Tamara-Kat/goit-js-hw-your-energy-TamaraK1.git](https://github.com/Tamara-Kat/goit-js-hw-your-energy-TamaraK1.git)
```
2. **Proje klasörüne gidin:**
```bash
cd goit-js-hw-your-energy-TamaraK1
```
3. **Bağımlılıkları yükleyin:**

```bash
npm install
```
4. **Geliştirme modunu başlatın:**

```bash
npm run dev
```
Ardından tarayıcınızda http://localhost:5173 adresini açın.

## Projeyi Derle (Build)
Üretim için optimize edilmiş bir proje sürümü oluşturmak için şunu çalıştırın:

```bash
npm run build
```
Tamamlanan dosyalar dist klasöründe olacaktır.
